# Portifolio
Portfólio pessoal com projetos e informações sobre minha carreira como desenvolvedor.

## Tecnologias Utilizadas
Este projeto foi construído utilizando as seguintes bibliotecas e ferramentas principais:

- React: Biblioteca JavaScript para construção da interface de utilizador.
- Tailwind CSS: Framework de CSS utility-first para estilização rápida e responsiva.
- Framer Motion: Biblioteca para criar animações complexas e transições suaves de componentes.
- React Anchor Link Smooth Scroll: Para navegação suave entre as secções da página através do menu e botões.
- React Hooks: Utilização de hooks como useState, useEffect e o custom hook useMediaQuery para gestão de estado e responsividade.
